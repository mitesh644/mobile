Technologies

HTML

HTML (HyperText Markup Language): The Skeleton. It defines the structure. It tells the browser where the headings, paragraphs, and images go.

CSS

CSS (Cascading Style Sheets): The Skin and Clothes. It handles the design. It’s how you choose colors, fonts, spacing, and layouts.

JavaScript

JavaScript (JS): The Brain. It adds interactivity. When you click a button and a menu pops up, or you see a countdown timer, that is JavaScript at work.

BOOTSTRAP 5

Bootstrap: A CSS framework. Instead of writing 100 lines of CSS to make a button look pretty and a layout look straight, you use Bootstrap’s "classes" to do it in seconds.

REACT.JS

React JS:Created by Meta (Facebook), this is a JavaScript library for building user interfaces. It uses "components" (reusable pieces of code), making it very efficient for complex sites like Instagram or Netflix.

Fully Responsive

Responsive isn't a coding language; it's a goal, You achieve this using CSS "Media Queries" or frameworks like Bootstrap, which is designed specifically to be "Mobile First."
